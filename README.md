# Retail Sales Data Analysis using Python

## Project Overview
This project performs end-to-end exploratory data analysis (EDA) on a retail sales dataset to uncover insights related to sales performance, profitability, customer behavior, and regional trends.

## Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Analyze sales and profit trends
- Identify loss-making products and regions
- Understand customer and segment performance
- Generate actionable business insights

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Plotly
- Jupyter Notebook

## Dataset
Superstore retail dataset containing transactional order data including sales, profit, customer, product, and region details.

## Key Analysis Performed
- Data Cleaning & Preparation
- Feature Engineering (Profit Margin, Shipping Delay, Time Features)
- Sales & Profit Analysis
- Category & Sub-category Performance
- Loss & Risk Analysis
- Customer Revenue Analysis
- Time-Series Sales Trend
- Discount vs Profit Analysis

## Key Insights
- Technology category generates highest profit.
- Excessive discounting reduces profitability.
- Few customers contribute majority of revenue.
- Some regions and sub-categories consistently generate losses.
- Sales show seasonal growth pattern.

## Business Recommendations
- Optimize discount strategy to protect margins
- Focus retention on high-value customers
- Improve performance in loss-making regions
- Re-evaluate low-profit product categories

## How to Run
1. Clone this repository
2. Install dependencies : pip install -r requirements.txt
3. Open notebook : jupyter notebook notebooks/Retail_Sales_Analysis.ipynb
